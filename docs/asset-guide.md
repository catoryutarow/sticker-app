# アセット追加・管理ガイド

シールの追加・削除・差し替え手順を説明します。

## ID命名規則

```
{パッケージID}-{シールID}
例: 001-001, 001-002, 002-001
```

- パッケージID: 3桁（001〜999）
- シールID: 3桁（001〜999）

## ディレクトリ構造

```
public/
  assets/
    stickers/     PNG画像ファイル
      001-001.png
      001-002.png
      ...
    audio/        MP3音源ファイル
      001-001.mp3
      001-002.mp3
      ...

src/
  config/
    stickerConfig.ts   シール定義（この1ファイルで管理）
```

## シールの追加手順

### 1. 設定ファイルにエントリを追加

`src/config/stickerConfig.ts` の `STICKERS` 配列に追加:

```typescript
export const STICKERS: StickerDefinition[] = [
  { id: '001-001', name: 'Star', nameJa: 'スター', color: '#FFD700' },
  { id: '001-002', name: 'Heart', nameJa: 'ハート', color: '#FF69B4' },
  // ↓ 新しいシールを追加（パッケージ001の7番目）
  { id: '001-007', name: 'Moon', nameJa: 'ムーン', color: '#C0C0C0' },
];
```

| プロパティ | 説明 | 例 |
|-----------|------|-----|
| `id` | パッケージID-シールID形式 | `'001-007'` |
| `name` | 英語名 | `'Moon'` |
| `nameJa` | 日本語名 | `'ムーン'` |
| `color` | テーマカラー（HEX） | `'#C0C0C0'` |

### 2. 画像ファイルを配置

`public/assets/stickers/{id}.png` に画像を配置:

```
public/assets/stickers/001-007.png
```

**画像仕様:**
- 形式: PNG（透過推奨）
- サイズ: 100x100px 以上推奨
- 正方形が望ましい

### 3. 音源ファイルを配置

`public/assets/audio/{id}.mp3` に音源を配置:

```
public/assets/audio/001-007.mp3
```

**音源仕様:**
- 形式: MP3
- 長さ: **16秒**（8小節 @ BPM120）
- サンプルレート: 44100Hz
- チャンネル: ステレオ

## シールの削除手順

1. `src/config/stickerConfig.ts` の `STICKERS` 配列から該当エントリを削除
2. `public/assets/stickers/{id}.png` を削除
3. `public/assets/audio/{id}.mp3` を削除

## シールの差し替え手順

ファイルを上書きするだけでOK:

- 画像差し替え: `public/assets/stickers/{id}.png` を上書き
- 音源差し替え: `public/assets/audio/{id}.mp3` を上書き

設定ファイルの変更は不要です。

## フォールバック動作

| 状況 | 動作 |
|------|------|
| PNG画像が存在しない | SVGフォールバックを表示 |
| MP3音源が存在しない | 音声なし（エラーにはならない） |

開発中はPNG/MP3がなくても動作します。

## 注意事項

- `id` は `{パッケージID}-{シールID}` 形式（例: `001-001`）
- パッケージIDとシールIDはそれぞれ3桁の数字
- 音源の長さは16秒（8小節）に統一すること（ループ同期のため）
- 画像・音源のファイル名は `{id}` と完全一致させること

## 確認方法

1. 開発サーバーを起動: `npm run dev`
2. シールパレットに新しいシールが表示されることを確認
3. シールをドラッグ＆ドロップして画像が表示されることを確認
4. 再生ボタンを押して音声が再生されることを確認
